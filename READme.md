## Parser Wildberiies

---
### Описание проекта
Это обычный парсер для сбора данных о товарах, 
продаваемых на популярном маркетплейсе Wildberries

### Установка 
1. Скачайте проект
2. Установите зависимости

### Использование

Запустите файл main.py

`python main.py`

Результаты будут сохранятся в data/output_data/
